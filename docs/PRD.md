# CT Generator v2 — PRD

## 비전

채팅 한 줄로 CT 카드를 만든다. **초안이 곧 최종**인 자판기 + 필요하면 수정/피드백도 가능한 구조.

### 디자인 원칙

1. **초안 완성도가 핵심** — 유저(카드상품 기획자 등 비디자이너)는 수정 없이 바로 쓸 수 있어야 함
2. **수정은 강제하지 않되, 길은 열어둠** — 채팅 수정, 캔버스 직접 조작 모두 가능
3. **AI가 알아서 품질 관리** — 명암비, 글자수, 구도 등 문제는 유저가 안 물어봐도 자동 감지/교정

│   │  메시지 입력...          │   │
│   └─────────────────────────┘   │
└─────────────────────────────────┘
              ↓ 입력 즉시 캔버스 모드 전환
┌──────────────┬──────────────────┐
│   Chat       │   Canvas         │
│              │                  │
│ 🤖 3가지 안  │  ┌───┐┌───┐┌───┐│
│ 만들었어요   │  │ 1 ││ 2 ││ 3 ││
│              │  └───┘└───┘└───┘│
│              │    ○   ●   ○    │
│              │                  │
│              │  ⚠️ AI 피드백     │
│              │  "안2 명암비 좋음" │
│              │                  │
│ ┌──────────┐ │  [다운로드]       │
│ │ 입력...   │ │                  │

```

- 첫 화면은 채팅 인풋만 (심플)
- 입력 즉시 **완성본 3안 동시 생성** + 캔버스 모드 전환
- 각 안은 문구/색상/배경처리까지 완성된 상태
- AI 피드백 자동 표시 (명암비, 가독성 등)
- 마음에 드는 안 선택 → 바로 다운로드 가능 (여기서 끝나도 OK)

### Step 2: 수정이 필요하면 (선택적)

**방법 A — 채팅으로 수정**

```
👤 "제목을 좀 더 짧게 바꿔줘"
🤖 수정안 3개 → 카드에 반영
```

**방법 B — 캔버스 직접 조작**

```
┌──────────────┬──────────────────┐
│   Chat       │   Canvas         │
│              │                  │
│              │  ┌────────────┐  │
│              │  │ [선택됨]    │  │
│              │  │ "3월 적금"  │──┼─→ 수정 팝오버
│              │  │             │  │   ┌──────────┐
│              │  └────────────┘  │   │ 대안 1    │
│              │                  │   │ 대안 2    │

- 텍스트 클릭 → 해당 필드 대안 3개 + 직접입력
- 이미지 드래그 → 크롭 위치 조정
- 배경 클릭 → 그라데이션/솔리드/없음 옵션

## AI 자동 피드백 (항상 작동)

유저가 요청하지 않아도 AI가 자동으로 품질 체크하고 피드백.

| 항목 | 체크 내용 | 대응 |
|------|-----------|------|
| 명암비 | WCAG AA 미달 | 텍스트색 BK↔WT 자동 전환 또는 경고 |
| 글자수 | 34byte 초과 | 자동 축약안 제시 |
| 구도 | 텍스트-이미지 겹침/가독성 | 배경처리(그라데이션) 자동 추가 |
| 이미지 | 해상도 부족, 비율 이상 | 경고 + 대안 제안 |

→ 피드백은 캔버스 하단에 비침습적으로 표시. "이 조합 괜찮습니다 ✓" 도 보여줌.

## 핵심 기능

### 1. 대화형 카드 생성

| 입력 | AI 동작 |
|------|---------|
| "3월 적금 이벤트 카드 만들어줘" | 완성본 3안 생성 (문구 + 색상 + 배경처리) |
| "좀 더 젊은 느낌으로" | 톤 변경한 3안 재생성 |
| "이 이미지 써줘" + 파일 첨부 | 이미지 배치 + 배경처리 자동 적용 |
| "아이콘으로 만들어줘" | SVG 아이콘 생성 + 주제부 배치 |

### 2. 멀티 수정안 (Variants)

- **첫 생성 시에도 항상 3안** (수정 요청 아니어도)
- 탭/스와이프로 비교
- "안 2가 좋아" 또는 그냥 탭 → 확정

```
  [ 안 1 ]  [ 안 2 ]  [ 안 3 ]
     ○         ●         ○
          [다운로드]
```

### 3. 캔버스 셀렉트 → AI 수정

카드의 특정 영역을 선택하면 해당 필드만 타겟팅하여 수정안 제시.

- **텍스트 선택**: 해당 필드(label/title/sub)의 대안 3개 제시
- **이미지 선택**: 크롭 위치 조정 / 다른 이미지 제안
- **배경 선택**: 그라데이션 방향/색상 변경안

### 4. 이미지 처리

- **큰 이미지**: AI 피사체 감지 → 최적 크롭 자동, 드래그로 미세 조정
- **작은 이미지/로고**: 주제부(카드 중앙)에 자동 배치
- **이미지 합성**: 여러 소스를 합성 (Gemini Nano/Banana/Pro)
- **아이콘 생성**: 텍스트 프롬프트 → SVG 아이콘

## AI 백엔드

### 텍스트 생성/수정
- Gemini API (문구 생성, 톤 조절, 피드백)
- 34byte 제한 자동 검증 후 재생성

### 이미지 처리
- Gemini Pro: 이미지 합성, 배경 생성
- Gemini Nano: 경량 이미지 분석 (피사체 감지, 색상 추출)

### 아이콘 생성
- SVG 아이콘 생성 (프롬프트 기반)

## 기술 구현 방향

### 상태 관리

```
ChatMessage[] ←→ CTVariant[] (3안)
                      ↕
               selectedIndex → CTContent (현재 선택된 안)
```

- 채팅 메시지와 카드 variants 양방향 동기화
- 각 variant는 독립적인 CTContent 스냅샷

### 캔버스 인터랙션

- CT 카드의 각 영역(label, title, sub, image, bg)을 히트 테스트 가능한 레이어로 분리
- 클릭/드래그 이벤트 → 해당 필드 식별 → AI에 컨텍스트 전달
- 수정안 팝오버는 선택된 영역 근처에 표시

### 레이아웃 전환

- 초기: `flex justify-center` (채팅 중앙)
- 캔버스 모드: `grid grid-cols-[400px_1fr]` (좌 채팅 / 우 캔버스)
- 트랜지션: Tailwind + CSS transition으로 부드러운 전환
